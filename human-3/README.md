# Human 3.0

The vision for AI-augmented humans at WDI.

## The Evolution

| Version | Description |
|---------|-------------|
| Human 1.0 | Early humans — survival, basic tool use |
| Human 2.0 | Modern humans — language, culture, technology |
| **Human 3.0** | AI-augmented humans — extended cognition, persistent memory |

## Core Principles

1. **AI should magnify everyone** — not just the top 1%
2. **Goal orientation over task orientation** — know your mission
3. **Continuous learning** — capture signals, reflect, improve
4. **Transparency** — agent reasoning and memory are observable
5. **Earned autonomy** — start supervised, graduate to independence

## Components

| Component | Purpose |
|-----------|---------|
| **TELOS** | Framework for articulating identity/consciousness |
| **Fabric** | AI augmentation patterns (240+ prompts) |
| **PAI** | Personal AI Infrastructure (Claude Code integration) |
